## Overview

AI Art Animator is a professional stroke-level drawing animation system that converts static images into realistic "artist drawing" videos.

## Features

- 6 artistic styles (Pencil, Watercolor, Oil, Charcoal, Anime, Advanced Sketch)
- Realistic stroke simulation with human-like jitter
- Multi-layer rendering (outline, detail, shading)
- Customizable animation parameters
- GUI and CLI interfaces
- Batch processing support
- Multiple export formats (MP4, GIF)
